# 叩命 (KouMing) — 产品规格书

> 版本: v1.0.0 | 更新: 2026-04-22 | 状态: 开发中

---

## 一、产品概述

### 名称
- 中文: 叩命
- 英文: KnockFate
- 日文: 運命を叩く
- 韩文: 운명을 두드리다

### 一句话描述
叩问深渊，深渊回响 —— 一个融合许愿、占卜、社交的灵性社区应用。

### 核心定位
- 情感驱动型微付费产品（类似直播打赏模式）
- 全球市场：中国为主，覆盖东亚文化圈（日韩），辐射全球
- 目标用户：18-35岁，对玄学/占卜/许愿有兴趣的年轻人

## 二、核心功能

### 2.1 深渊许愿池（首页）
- 用户输入愿望文字（最多50字），投币许愿
- 愿望沉入池底动画（金币飞行+水花+涟漪）
- 每日免费3次叩问
- 池中显示其他用户愿望气泡（根据灯数分级显示）
- 热门愿望排行

### 2.2 池灵算卦（核心变现）
- 许愿后，池灵AI对话引导
- **第一次免费**：完整卦象+解读+建议+相似愿望数据
- **后续收费**：¥6/次 或 ¥18/月
- 5种卦象分类：学业(study)、健康(health)、感情(love)、财运(money)、其他(default)
- AI生成个性化解读（使用 Gemini API 免费额度）
- 卦象：渐卦、师卦、咸卦、丰卦、临卦

### 2.3 智能捞愿（社交发现）
- 每日5次免费打捞
- 70%语义匹配 + 30%随机惊喜
- 点灯功能：为他人愿望点灯祈福
- 点灯动画效果

### 2.4 供奉商店
- 供奉物品：线香(¥1)、莲花灯(¥2)、河灯(¥3)
- 天命签：¥6/签（上上签/上签/中签/中下签/下签）
- 还愿仪式：¥3.6（愿望到期后还愿+烟花庆祝）

### 2.5 愿望胶囊
- 每个愿望自动生成胶囊
- 不同类型有不同等待期（学业10月/感情6月/财运8月/其他3月）
- 到期后可还愿（烟花庆祝动画）

### 2.6 个人中心
- 我的愿望列表
- 功德系统：等级+进度条
- 徽章系统：初愿、叩命达人、点灯人、灯火阑珊、功德等级徽章
- 全局许愿计数器

## 三、技术架构

### 3.1 技术栈
- **框架**: Flutter 3.29+ (Dart)
- **平台**: iOS + Android + Web
- **AI服务**: Google Gemini API (免费额度) + 阿里云百炼 (国内备选)
- **后端**: Firebase (全球) / 云开发 (国内)
- **支付**: 
  - 中国: 微信支付 + 支付宝
  - 全球: Stripe + Google Play Billing + Apple IAP
- **i18n**: flutter_localizations + intl
- **状态管理**: Riverpod
- **本地存储**: Hive

### 3.2 目录结构
`
lib/
  main.dart
  l10n/                    # 自动生成的国际化文件
  app.dart                 # App入口+路由
  features/
    pool/                  # 深渊许愿池
    fortune/               # 池灵算卦
    shop/                  # 供奉商店
    capsule/               # 愿望胶囊
    profile/               # 个人中心
  shared/
    theme/                 # 主题+颜色+字体
    widgets/               # 通用组件
    models/                # 数据模型
  services/
    ai_service.dart        # Gemini/百炼 API
    wish_service.dart      # 愿望CRUD
    payment_service.dart   # 支付处理
    merit_service.dart     # 功德系统
    i18n/                  # 翻译文件
      zh.json
      en.json
      ja.json
      ko.json
`

### 3.3 AI 服务设计
- **主用**: Google Gemini 2.5 Flash（免费额度，全球可用）
  - 代理: 用户设备直连（需翻墙用户）/ Cloudflare Worker 中转
- **国内备选**: 阿里云百炼 qwen-image-2.0-pro
- **功能**: 愿望分类、卦象解读生成、语义匹配
- **限制**: 仅在算卦和捞愿时调用，减少API消耗

### 3.4 离线能力
- 核心功能（许愿、点灯、查看胶囊）可离线使用
- 算卦和AI捞愿需网络
- 数据本地缓存，联网后同步
